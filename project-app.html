Companion Destktop Application

Created in Visual Studio, this C++ application is an on-going and growing project.

This application was created with the intent to increase productivity as well as reduce mistakes made by users. Originally the application started out as a Note Template application. Over a few years it grew into a full-companion with Email Templates, Note Templates, Calculator and Quizzes.

The application is used by over 100 users daily to follow scripting, confirm all steps are taken appropriately and to make a lot of day to day work easier. Productivity increased by almost 80% after the first year. Enhancements continue to further develop the application to keep up with business growth.

  • 58,000 lines and counting of C++ Code
  • Connections with Microsoft Office Suites (Outlook, Excel etc)
  • Executable uploaded to Shared Network for ease of access and automatic updates when program is launched
  • Daily use application for 100s of users
  • Built in email for bugs/fixes and enhancements
  • Note template with on-screen queues, reminders and pop-ups.
  • Email template which loads all information into a new email connected with Outlook.
  • Payment Calculator which loads pricing from excel/query and calculates deductibles, balances and payments.
  • Quizzes which can be loaded for users to take on-screen questions, visual checks and reporting sent via Outlook.
